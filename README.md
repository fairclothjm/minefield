# minefield
Minefield - a minesweeper clone built with pygame.

<img src="minefield1.png" alt="Minefield" width=300/>
<img src="minefield2.png" alt="Minefield" width=300/>

## installation
``` bash
git clone https://github.com/fairclothjm/minefield.git
```
``` bash
paver setup
```

## play
``` bash
paver run
```
